Exámenes Médicos

Ofrecemos una variedad de exámenes médicos para garantizar tu salud:

  • Exámenes de Sangre: Incluyen análisis completos de sangre para detectar diversas condiciones.
  • Exámenes de Imágenes: Rayos X, ecografías, resonancias magnéticas y tomografías.
  • Exámenes de Diagnóstico: Pruebas específicas para detectar enfermedades como diabetes, colesterol.
  • Exámenes de Función: Pruebas para evaluar el funcionamiento de órganos como el hígado, riñones, entre otros.
